# Инструкции для агентов
- Все создаваемые или редактируемые текстовые материалы (документация, ответы
  в PR, описания изменений) должны быть написаны на русском языке.
- При отсутствии особых оговорок придерживайтесь существующих соглашений
  репозитория и обновляйте спецификацию бота в каталоге `doc/` также на
  русском языке.
- Допускается исключительно технологический стек на языке программирования
  Rust. Никакие другие языки или платформы в репозитории не используются и не
  предлагаются.
- Перед завершением задачи обязательно запускайте:
  - проверку форматирования (например, `cargo fmt --check`);
  - сборку/проверку (`cargo check`);
  - линтер (`cargo clippy -- -D warnings`);
  - тесты (`cargo test`).
  Команды следует выполнять для всех доступных Rust-компонент. При возникновении
  ошибок необходимо внести исправления до успешного завершения.
